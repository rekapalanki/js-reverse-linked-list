Reverse a linked list

What is a linked list?

A linked list is a list of elements with each items storing a pointer to the next one.

Imagine the linked list like a chain of paperclips. You can add new items to each end or to the middle by disconnecting the chain.

The element of a linked list is called a node. The first element is the head and the last element is the tail.

Uses:

Stacks and queues need fast operations at the end, so linked lists are ideal to deal with.

Sample linked list implementation

Usually, programming languages don't include an implementation of a linked list.

Here's a sample linked list node class:

class LinkedListNode {
    constructor(value) {
        this.value = value;
        this.next = null;
    }
}

Assuming that we've created the above class we can create sample list elements.

    const a = new LinkedListNode(5);
    const b = new LinkedListNode(10);
    const c = new LinkedListNode(12);
    const d = new LinkedListNode(33);
    const e = new LinkedListNode(400);
    
    a.next = b;
    b.next = c;
    c.next = d;
    d.next = e;

Iterating through the original nodes

To iterate through our original list items we write a function. 

function listLinkedNodes(headOfList) {
    const head = headOfList;
    let list = [head.value];
    let element = head;
    while (element.next) {
        element = element.next;
        list.push(element.value);
    }
    return list;
}

Return the new head of the reversed list

Creating a function that finds the last element of the list and returns its value.

function reverseListHead(headOfList) {
    const originalHead = headOfList;
    let nextElement = originalHead.next;
    let previousElement = nextElement;
    while (nextElement) {
        previousElement = nextElement;
        nextElement = nextElement.next;
    }
    return(previousElement.value);
}
