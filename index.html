Reverse a linked list

What is a linked list?

A linked list is a list of elements with each items storing a pointer to the next one.

Imagine the linked list like a chain of paperclips. You can add new items to each end or to the middle by disconnecting the chain.

The element of a linked list is called a node. The first element is the head and the last element is the tail.

Usually, programming languages doesn't include an implementation of a linked list.

Here's a sample linked list node class.

class LinkedListNode {
    constructor(value) {
        this.value = value;
        this.next = null;
    }
}

Assuming that we've created the above class we can create sample list elements.

const a = new LinkedListNode(5);
const b = new LinkedListNode(10);
const c = new LinkedListNode(12);
const d = new LinkedListNode(33);
const e = new LinkedListNode(400);
